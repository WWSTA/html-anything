使用说明

三步把你的文字变成读者想看的成品

1

挑一张模板

根据你手上的内容类型（长文本、数据、PPT、社交素材），选最合适的视觉风格。每张模板都附了 SKILL.md 详细说明和 example.html 成品示例。

  • 按「分类」快速缩小范围
  • 用「场景」匹配你的工作场景
  • 按标签精确查找具体风格
2

看模板详情

点击任意模板卡片查看完整信息，包括配色方案、字体栈、布局结构、硬性设计约束，以及最关键的示例输入 / 输出。

  • 阅读视觉签名（颜色、字体、间距）
  • 看 example.html 成品效果
  • 了解适合的输入内容格式
3

生成你自己的 HTML

把你的真实内容（Markdown / CSV / JSON / 纯文本）交给 AI，套用选中模板的约束即可生成可直接分享的成品 HTML。

  • 不编造数据，只用用户真实输入
  • 单文件 HTML，不依赖外部资源
  • 支持打印为 PDF、导出 PNG
💡 小贴士： 拿不定主意时，通用文档优先用 doc-kami-parchment，PPT 优先用 deck-swiss-international，海报类优先用 magazine-poster。
